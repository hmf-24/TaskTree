## 3. 核心模块设计

### 3.1 认证模块

```python
# JWT Token结构
{
  "sub": "user_id",
  "email": "user@example.com",
  "exp": 1712812800,
  "iat": 1710201600
}

# 密码加密：bcrypt
# Token有效期：7天（可配置）
```

### 3.2 项目管理模块

```
ProjectService
├── create_project()      # 创建项目
├── update_project()      # 更新项目
├── delete_project()      # 删除项目
├── list_projects()       # 项目列表
├── archive_project()     # 归档项目
└── get_project()         # 获取项目详情
```

### 3.3 任务树模块

```
TaskService
├── create_task()         # 创建任务
├── update_task()        # 更新任务
├── delete_task()        # 删除任务
├── move_task()          # 移动任务（调整层级/顺序）
├── get_task_tree()      # 获取任务树
├── batch_create()       # 批量创建
└── update_progress()    # 更新进度
```

**任务树数据结构**：
```typescript
interface TaskTreeNode {
  id: number;
  name: string;
  children: TaskTreeNode[];
  // ...其他属性
}
```

### 3.4 依赖关系模块

```
DependencyService
├── create_dependency()  # 创建依赖
├── delete_dependency()  # 删除依赖
├── check_dependency()   # 检查依赖（前置任务是否完成）
└── get_dependencies()   # 获取依赖关系
```

### 3.5 数据导入导出

```
ExportService
```

## 6. 部署方案

### 6.1 Docker Compose

```yaml
version: '3.8'

services:
  backend:
    build: ./backend
    ports:
      - "8000:8000"
    volumes:
      - backend-data:/app/data
    environment:
      - DATABASE_URL=sqlite+aiosqlite:///app/data/tasktree.db

  frontend:
    build: ./frontend
    ports:
      - "80:80"
```

**启动命令**：
```bash
docker-compose up -d
```

### 6.2 Nginx 反向代理

前端 Nginx 配置实现：
- 静态资源服务
- API 反向代理到后端
- 静态资源缓存

### 6.3 环境变量

```env
# .env
SECRET_KEY=your-secret-key-change-in-production
DEBUG=false
DATABASE_URL=sqlite+aiosqlite:///./data/tasktree.db
```

## 8. 性能优化策略

| 场景 | 优化方案 |
|------|----------|
| 大量任务渲染 | 虚拟滚动（react-window） |
| 图片加载 | 懒加载 + 压缩 |
| API请求 | 请求缓存 + 乐观更新 |
| 实时同步 | 节流 + 增量更新 |
| 数据库查询 | 索引优化 + 懒加载 |
