# 项目计划 v1.0

> [!info] 项目概览
> TaskTree 任务树 — 全栈跨平台应用，React + TypeScript + Tauri + Python + FastAPI，开发周期约8周。

> [!tip] 关键技术决策
> - 前后端分离 + Tauri 桌面打包，一套代码多端运行
> - 任务树使用自引用设计，支持无限层级
> - 多视图并行：树形 / 看板 / 甘特图 / 依赖图

## 风险管理

> [!warning] 关键风险识别
> 以下风险可能影响项目进度，需提前关注和应对

| 风险 | 可能影响 | 应对措施 |
|------|----------|----------|
| 大数据量性能下降 | 高 | 虚拟滚动、懒加载 |
| 实时同步数据冲突 | 中 | 乐观锁机制 |
| 甘特图组件功能不足 | 中 | 提前评估，必要时二次开发 |
| 团队协作开发延期 | 中 | 优先核心功能，非核心延后 |
