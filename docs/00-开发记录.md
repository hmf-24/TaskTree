## 阶段二/三：高级视图与扩展功能开发
**日期**: 2026-04-11 (第2天 上午/下午)

### 🚀 核心完成内容
1. **多维度视图系统重构**
   - **任务树拖拽**: 引入 `@dnd-kit` 实现复杂的无限层级无缝拖拽重排与父子节点转移。
   - **看板视图**: 提供按状态划分的看板展示模式 (`KanbanView.tsx`)。
   - **列表与甘特图**: 增加 AntD Table 处理数据报表；接入 `gantt-task-react` 处理任务时间线。
   - **依赖关系图**: 引进 `ReactFlow` 实现任务依赖链路的高级可视化。

2. **任务与标签系统强化**
   - **侧边控制台**: 开发 `TaskDetailDrawer.tsx` 抽屉，实现任务各属性的实时内联编辑。
   - **评论与集成**: 侧边台增加评论与 @ 推送逻辑 (`CommentList.tsx`)。
   - **多阶标签**: 完成全局项目级别彩色标签（跨任务交叉筛选功能底层预留）。

3. **进阶数据流转**
   - **消息轮询**: 建立基于 HTTP 轮询的 `NotificationPanel.tsx` 系统及相应的数据库模型。
   - **多格式导出与恢复**: 后端 `openpyxl` 及 JSON 处理完成，支持向用户提供 Markdown 报告、Excel 表格、JSON 完整树形数据的一键导出导入互转体系。
