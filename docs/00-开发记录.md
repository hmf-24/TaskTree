## 阶段五：智能提醒系统与大模型集成
**日期**: 2026-04-21

### 🔔 核心完成内容
1. **多厂商大模型支持**
   - 新增统一 `LLMService`，支持 Minimax、OpenAI、Anthropic 三家主流厂商。
   - 前端设置页面可选择大模型提供商和具体模型版本，用户可配置个人的 API Key（加密存储）。
2. **多维度智能分析与调度**
   - 逾期检测、进度落后、依赖解除、团队负荷分析、风险预测等多维度评估。
   - 自然语言规则解析器（理解“每天/每周/周末”、“下午3点”等意图）。
   - 自动生成任务标签、优先级、分类建议及具体的任务规划建议。
3. **钉钉与通知中心集成**
   - 集成钉钉机器人 Webhook 推送（支持签名验证）。
   - 定时任务调度器每30分钟检查用户任务，并包含已读回执、消息去重（6小时内不重复）、状态持久化。
   - 调度面板支持发送数量、已读率统计及"立即提醒"手动触发功能。
