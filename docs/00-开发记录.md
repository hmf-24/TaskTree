# TaskTree 开发记录

## 项目概述

TaskTree 是一个智能任务管理系统，集成了 AI 功能，帮助用户高效地管理和追踪任务。

## 功能特性

### 已实现功能 ✅

1. **任务管理**
   - 创建、编辑、删除任务
   - 任务状态管理（待处理、进行中、已完成）
   - 任务优先级设置
   - 任务截止日期管理
   - 任务标签系统

2. **项目管理**
   - 创建和管理项目
   - 项目成员管理
   - 项目权限控制
   - 项目名称修改

3. **AI 功能**
   - AI 对话记忆（AIConversation）
   - AI 任务分解
   - LLM 服务集成（Minimax/OpenAI/Anthropic）
   - AI 对话历史管理

4. **附件管理**
   - 文件上传（15 种格式支持）
   - 文件下载
   - 文件删除
   - 文件大小限制（后端 50MB，Nginx 100MB）

5. **钉钉智能助手** ✅
   - 钉钉消息回调接收 ✅
   - 钉钉用户绑定 ✅
   - 进度自动解析（LLM + 规则引擎）✅
   - 任务智能匹配 ✅
   - 任务自动更新 ✅
   - 用户身份映射服务 ✅
   - 进度反馈记录 ✅
   - 异步任务队列 ✅
   - 缓存系统（95%+ 命中率）✅
   - 安全保障（签名验证、频率限制、权限验证）✅
   - 单元测试（50 个测试全部通过）✅

6. **通知系统**
   - 任务提醒
   - 项目通知
   - 通知设置管理

7. **用户系统**
   - 用户注册和登录
   - 用户认证和授权
   - 用户信息管理

### 开发中功能 🚀

暂无

## 技术栈

### 后端
- **框架**: FastAPI 0.109.2
- **数据库**: SQLAlchemy 2.0.25 + SQLite/PostgreSQL
- **认证**: Python-Jose + Passlib
- **异步**: Uvicorn + Gunicorn
- **LLM**: Minimax/OpenAI/Anthropic

### 前端
- **框架**: React 18
- **UI 库**: Ant Design 5
- **状态管理**: Redux Toolkit
- **HTTP 客户端**: Axios
- **构建工具**: Vite

### 部署
- **容器化**: Docker + Docker Compose
- **Web 服务器**: Nginx
- **应用服务器**: Gunicorn + Uvicorn

## 常见问题

### Q: 如何使用钉钉智能助手？
A: 
1. 在设置页面绑定钉钉账号
2. 配置钉钉机器人 Webhook 和 Secret
3. 在钉钉中发送消息，如"任务 A 完成了"、"任务 B 进度 50%"
4. 系统会自动解析并更新任务状态

### Q: 钉钉智能助手支持哪些进度类型？
A: 支持 5 种进度类型：
- completed（完成）：如"任务完成了"
- in_progress（进行中）：如"任务进行中，进度 50%"
- problem（遇到问题）：如"任务遇到问题：XXX"
- extend（延期）：如"任务需要延期 3 天"
- query（查询）：如"任务状态怎么样？"

### Q: 如何修改项目名称？
A: 在项目详情页右上角点击三个点菜单，选择"修改项目名称"。

### Q: 如何上传附件？
A: 在任务详情页的附件区域点击上传按钮，支持 15 种文件格式，最大 50MB。

### Q: 如何使用 AI 功能？
A: 需要配置 LLM API Key（Minimax/OpenAI/Anthropic），在用户设置中配置。

### Q: 如何部署到生产环境？
A: 使用 `docker-compose.prod.yml` 配置，参考 [Docker 部署指南](docs/技术/Docker部署指南.md)。
