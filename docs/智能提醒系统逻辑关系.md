## 配额计算逻辑 ✅

```
每日配额计算:

自动提醒配额:
  today_auto_count = NotificationLog
    .where(is_manual == false)
    .where(sent_at >= today_start)
    .count()

if today_auto_count >= daily_limit:
    skip()  # 跳过本次自动提醒
    return False

手动触发:
  always allowed  # 不检查配额
  is_manual = true
  return True  # 发送成功
```

---

## 返回值约定

| 返回值 | 含义 |
| :--- | :--- |
| `True` | 操作成功发送了提醒 |
| `False` | 操作失败或已达配额 |
| `None` | 不需要提醒 |
