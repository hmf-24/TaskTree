## 配额计算逻辑

```
每日配额计算:

自动提醒配额:
  today_auto_count = NotificationLog
    .where(is_manual == false)
    .where(sent_at >= today_start)
    .count()

if today_auto_count >= daily_limit:
    skip()  # 跳过本次自动提醒

手动触发:
  always allowed  # 不占用自动提醒配额
  is_manual = true
```
