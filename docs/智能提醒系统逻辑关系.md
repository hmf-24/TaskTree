# TaskTree 智能提醒系统逻辑关系文档

## 系统整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                      前端 (React)                          │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────────┐ │
│  │  设置页面   │  │ 项目详情页  │  │  通知面板      │ │
│  │  Settings   │  │  Project    │  │  Notification │ │
│  └──────┬──────┘  └──────┬──────┘  └───────┬──────────┘ │
└─────────┼───────────────┼────────────────┼────────────────┘
          │               │                │
          ▼               ▼                ▼
┌─────────────────────────────────────────────────────────────┐
│                    API 网关 (FastAPI)                        │
│  ┌──────────────────────────────────────────────────────┐  │
│  │ /api/v1/notification_settings                │  │
│  │  - GET/POST /settings     提醒设置            │  │
│  │  - GET /logs              发送记录            │  │
│  │  - POST /callback/{id}    已读回执            │  │
│  │  - POST /trigger          手动触发            │  │
│  │  - GET /stats             统计报表            │  │
│  │  - POST /intent/parse      意图解析            │  │
│  │  - POST /tasks/auto-classify  任务分类        │  │
│  │  - GET /rules/template   规则模板            │  │
│  └──────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────────────────┐
│                   大模型服务 (LLMService)                    │
│  ┌───────────────┐    ┌───────────────┐    ┌───────────────┐  │
│  │   Minimax    │    │    OpenAI     │    │  Anthropic   │  │
│  └───────────────┘    └───────────────┘    └───────────────┘  │
│  - analyze_tasks()        多维度任务分析                    │
│  - parse_user_intent()     用户意图解析                      │
│  - auto_classify_tasks()   任务自动分类                        │
│  - analyze_complexity()   任务复杂度分析                    │
│  - generate_suggestions()  任务建议生成                      │
└─────────────────────────────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────────────────┐
│               调度器 (ReminderScheduler)                    │
│  每30分钟执行一次:                                           │
│  1. 获取所有启用提醒的用户                                  │
│  2. 检查自动提醒的每日发送上限                           │
│  3. 调用 LLMService.analyze_tasks()                         │
│  4. 如需提醒 → 调用钉钉服务                               │
│  5. 记录发送日志 (is_manual=false)                        │
└─────────────────────────────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────────────────┐
│                 钉钉服务 (DingTalkService)                    │
│  - send_message()        发送Markdown消息                   │
│  - send_task_reminder()  发送单个任务提醒                     │
│  - send_batch_reminder() 发送批量任务提醒                     │
│  - _generate_sign()     生成签名（安全验证）                 │
└─────────────────────────────────────────────────────────────┘
```

## 数据模型关系

```
┌─────────────────────────────────────────────────────────────┐
│                       User (用户表)                         │
│  id, email, nickname, password_hash, created_at             │
└─────────────────────────────────────────────────────────────┘
          │
          │ 1:1
          ▼
┌─────────────────────────────────────────────────────────────┐
│              UserNotificationSettings                       │
│  dingtalk_webhook, dingtalk_secret                         │
│  llm_provider, llm_api_key, llm_model, llm_group_id        │
│  analysis_config, rules, enabled, daily_limit               │
└─────────────────────────────────────────────────────────────┘
          │
          │ 1:N
          ▼
┌─────────────────────────────────────────────────────────────┐
│                    NotificationLog                       │
│  user_id, task_id, project_id                            │
│  message_id, message_content, is_read, read_at, sent_at     │
│  is_manual  ← 区分手动/自动触发                         │
└─────────────────────────────────────────────────────────────┘
```

## 配额计算逻辑

| 触发方式 | 是否计入配额 | 说明 |
| :--- | :--- | :--- |
| 自动调度 | ✅ 是 | 每日上限由daily_limit控制 |
| 手动触发 | ❌ 否 | 不占用自动提醒配额 |

---

## 返回值约定

| 返回值 | 含义 |
| :--- | :--- |
| `True` | 发送成功 |
| `False` | 失败/已达配额 |
| `None` | 不需要提醒 |
